# MyWiki MVP 로드맵

> 기준일: 2026-07-14

## 완료 기준

서로 다른 두 회원의 위키가 기본적으로 완전히 분리되고, 문서 소유자는 선택한 문서만 다른 회원에게 `viewer` 또는 `editor` 권한으로 공유·회수할 수 있어야 한다. 회원은 권한이 있는 문서만 검색할 수 있고, 소유자는 변경 이력을 확인하고 이전 버전을 새 revision으로 복구할 수 있어야 한다.

## 1단계 — 기반과 인증 (7월 14일~16일)

- [x] Flask 애플리케이션 팩토리와 Blueprint 구조
- [x] 환경 설정, PostgreSQL/SQLite 연결, Alembic
- [x] User·Wiki·인증 토큰 모델
- [x] 이메일·비밀번호 회원가입과 로그인
- [x] 가입 시 개인 위키 하나 자동 생성
- [x] 이메일 인증과 비밀번호 재설정의 콘솔·범용 SMTP 백엔드
- [x] 기본 레이아웃, 오류 화면, 상태 확인 URL
- [x] 인증 및 회원 데이터 격리 테스트
