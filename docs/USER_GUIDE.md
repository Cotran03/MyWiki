# MyWiki 사용 설명서

## 계정 만들기

1. 대문에서 `시작하기`를 선택합니다.
2. 영문 소문자 기반 사용자명, 표시 이름, 이메일, 12자 이상 비밀번호를 입력합니다.
3. 콘솔 메일 설정에서는 서버 터미널의 `DEV MAIL` 인증 URL을 엽니다. SMTP 설정에서는 받은편지함의 MyWiki 인증 메일을 엽니다.
4. 인증 완료 후 로그인합니다.

사용자명은 문서 URL에 포함되며 MVP에서는 변경할 수 없습니다.

가입하면 개인 위키와 다음 안내 문서가 자동으로 만들어집니다.

- `MyWiki 사용법`: 문서 작성·검색·공유·이력·휴지통 등 주요 기능 안내
- `Markdown 사용법`: MyWiki에서 지원하는 제목·강조·목록·링크·인용·코드 문법 안내

## 문서 작성

1. 상단의 `새 문서`를 선택합니다.
2. 제목과 Markdown 본문을 입력합니다.
3. 태그는 쉼표로 구분합니다.
4. `미리보기`를 선택해 저장될 본문 모양을 확인합니다.
5. 필요하면 `작성`으로 돌아가 내용을 고치거나 편집 요약을 적고 저장합니다.

`Ctrl+S`로 저장할 수 있습니다. 미리보기와 저장된 문서는 서버에서 같은 Markdown 처리 과정을 거칩니다. Markdown 원시 HTML과 스크립트는 보안을 위해 실행되지 않습니다.

## 검색

상단 검색창이나 사이드바의 `검색`을 사용합니다. 내 문서와 공유받은 문서 중 권한이 있는 문서의 제목·현재 본문·태그만 검색됩니다. `/` 키로 상단 검색창에 초점을 이동할 수 있습니다.

## 문서 공유

1. 소유한 문서를 엽니다.
2. `공유`를 선택합니다.
3. 이메일 인증을 마친 기존 회원의 사용자명을 입력합니다.
4. `열람자` 또는 `편집자`를 선택합니다.

- 열람자: 문서 읽기만 가능
- 편집자: 읽기와 수정 가능
- 소유자: 수정·공유·삭제·이력 복구 가능

공유 화면에서 `회수`를 선택하면 해당 회원의 접근은 즉시 중단됩니다.

## 변경 이력과 복구

문서 소유자는 `이력`에서 모든 revision을 볼 수 있습니다. 과거 revision의 `이 버전을 새 revision으로 복구`를 선택하면 과거 기록을 지우지 않고 새 revision이 만들어집니다.

공유 전 내용에 민감정보가 남아 있을 수 있으므로 viewer/editor에게는 이력이 공개되지 않습니다.

## 동시 편집 충돌

편집 화면을 연 뒤 다른 사용자가 먼저 저장했다면 이전 화면의 저장은 409 충돌로 거절됩니다. 최신 문서를 다시 열고 변경 내용을 합쳐야 합니다. 마지막 저장으로 다른 사용자의 내용을 자동 덮어쓰지 않습니다.

## 휴지통

소유자가 `휴지통`을 선택하면 문서는 30일 보관 상태가 됩니다.

- 공유 사용자는 즉시 접근할 수 없습니다.
- 검색 결과에서 제외됩니다.
- 소유자는 휴지통에서 복구할 수 있습니다.

## 화면 테마와 키보드

상단의 `◐` 버튼으로 `자동 → 라이트 → 다크`를 전환합니다.

- `/`: 검색창으로 이동
- `Ctrl+S`: 문서 편집 저장
- `Tab`, `Shift+Tab`: 링크와 컨트롤 이동
- 본문 바로가기: 페이지 첫 포커스 링크
