# MyWiki MVP 요구사항

> 상태: 승인됨
> 승인일: 2026-07-14
> 목표일: 2026-07-31(KST)

## 제품 정의

MyWiki는 회원마다 하나의 비공개 개인 위키를 제공하는 Markdown 기반 지식 관리 서비스다. 문서는 소유자가 지정한 기존 회원에게만 `viewer` 또는 `editor` 권한으로 공유할 수 있다.

## 사용자 역할

- **회원:** 자신의 개인 위키와 문서를 관리한다.
- **viewer:** 공유받은 활성 문서를 열람할 수 있다.
- **editor:** 공유받은 활성 문서를 열람하고 수정할 수 있다.
- **문서 소유자:** 문서 열람·수정·공유·삭제·복구와 이력 관리를 할 수 있다.
- **플랫폼 관리자:** 회원 계정과 서비스 상태를 관리하지만 비공개 본문 열람 권한은 자동으로 받지 않는다.

## 필수 기능

### 인증

- 이메일·비밀번호 회원가입, 로그인, 로그아웃
- 이메일 인증과 비밀번호 재설정
- 회원가입 완료 시 개인 위키 하나 자동 생성
- 비활성 계정 로그인 차단

### 문서

- Markdown 문서 작성·열람·수정
- 제목 변경과 무관한 UUID 기반 고정 URL
- 태그와 최근 문서 목록
- 원시 HTML을 허용하지 않는 안전한 Markdown 렌더링
- 수정할 때마다 불변 revision 생성
- 선택형 편집 요약과 기본 작업 요약 자동 생성

### 공유와 권한

- 기존 회원을 사용자명으로 찾아 문서 공유
- `viewer` 또는 `editor` 권한 부여·변경·회수
- 문서 소유자만 공유와 권한 변경 가능
- 무관한 회원에게 문서 존재와 검색 결과를 노출하지 않음
- 문서 이력 열람·비교·복구는 소유자만 가능

### 검색

- 제목·현재 본문·태그 부분 검색
- 소유하거나 공유받은 활성 문서만 결과에 포함

### 삭제와 복구

- 삭제 시 즉시 제거하지 않고 30일 휴지통으로 이동
- 휴지통 문서는 소유자만 확인·복구
- 휴지통 이동 중 공유 접근 중단
- 영구 삭제는 후속 관리 작업으로 수행

### 충돌 처리

- 편집 시작 revision과 저장 시점 revision이 다르면 저장하지 않음
- 사용자에게 409 충돌 안내와 최신 문서 재확인 경로 제공

### 운영과 품질

- `/health/live`, `/health/ready` 제공
- 인증, 권한, 문서 CRUD, 검색, revision, XSS에 대한 자동 테스트
- Windows 개발과 Linux 배포 설명서
- 환경 변수 기반 비밀 값 관리

## 스트레치 기능

- 즐겨찾기
- 라이트·다크·자동 테마
- 내부 문서 링크 선택기
- JPG·PNG·WebP 이미지 업로드(파일당 최대 5MB)

## 비기능 요구사항

- 모든 화면을 키보드로 사용할 수 있어야 한다.
- 기본 목표는 WCAG 2.2 AA다.
- 초기 용량 가정은 회원 100명, 동시 사용자 20명, 문서 10,000개, 첨부 10GB다.
- 운영 데이터베이스는 PostgreSQL을 사용한다.
- 문서·검색·revision·첨부의 권한 검사는 한 중앙 정책을 사용한다.
- 감사 로그에는 문서 본문, 비밀번호, 토큰을 기록하지 않는다.
- 외부 공개 전 HTTPS를 적용한다.

## 수용 테스트

1. 사용자 A와 B가 가입하면 각각 정확히 하나의 위키가 생성된다.
2. A와 B는 공유하지 않은 서로의 문서 URL·목록·검색 결과에 접근할 수 없다.
3. A가 B에게 viewer 권한을 주면 B는 해당 문서만 읽고 수정할 수 없다.
4. A가 B의 권한을 editor로 바꾸면 B는 문서를 수정할 수 있지만 공유·삭제·이력 복구는 할 수 없다.
5. A가 공유를 회수하면 B는 문서와 검색 결과에 다시 접근할 수 없다.
6. 문서를 수정할 때마다 revision이 하나 증가하고 과거 revision은 변경되지 않는다.
7. 과거 revision 복구는 새 revision을 만들며 이후 이력을 삭제하지 않는다.
8. 오래된 편집 폼 저장은 409로 거절된다.
9. Markdown에 입력한 스크립트가 실행 가능한 HTML로 출력되지 않는다.
10. 휴지통 문서는 소유자 외의 사용자에게 노출되지 않는다.
